Java数据类型分为基本类型和引用类型，具体见图片

前者表明Java并非纯OO语言，后者是其OO特性的体现

基本数据类型

布尔型 boolean，仅有两个值：true false, 占 1 位

其余是数值型，分为定点和浮点

定点有 char(2个字节) 、 byte(1个字节) 、 short(2个字节) 、 int(4个字节) 、 long(L)(8个字节)

浮点有 float(F)(4个字节) 和 double(D)(8个字节)

附1：小括号内字母附在数字后面，显式声明该值类型

附2：void，并非一种类型，只是标明方法等无返回值

验证代码

public class TestDataTypeSize {
    public static void main(String[] args) {
        System.out.println("char:"+Character.SIZE);
        System.out.println("byte:"+Byte.SIZE);
        System.out.println("short:"+Short.SIZE);
        System.out.println("int:"+Integer.SIZE);
        System.out.println("long:"+Long.SIZE);
        System.out.println("float:"+Float.SIZE);
        System.out.println("double:"+Double.SIZE);
    }

}

输出：
char:16
byte:8
short:16
int:32
long:64
float:32
double:64

引用数据类型

常见的有字符串，数组，接口，抽象类和类

包装类

常见基本类型都有对应对象类，后者封装了前者的一些信息

如：该类型的最值

依次为：

Boolean Character Byte Short Integer Long Float Double

封装类和对应基本类型之间可以自动转换，称为自动拆箱（前者转为后者）或自动装箱
