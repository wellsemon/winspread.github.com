CentOS 7 中安装 tomcat

下载 Tomcat

Wget 下载 Tomcat

Tomcat 官网中找到指定版本 Tomcat rpm 的 url

使用 wget url 下载 rpm ，下载文件位于当前目录

可用下列命令查看

[root@localhost ~]# ls -l *tomcat*

安装 Tomcat

创建 Tomcat 的目录

[root@localhost ~]# mkdir /usr/local/tomcat

在 Tomcat 下载文件所处目录中，执行解压命令

[root@localhost ~]# tar czvf tomcat-file-name /usr/local/tomcat

其中，tomcat-file-name 是下载的文件名

配置 Tomcat

配置 Tomcat 环境变量

[root@localhost ~]# vi /etc/profile

在文件中找 export PATH …

在其后，增加如下内容

export CATALINA_HOME=/usr/local/tomcat/apache-tomcat-7.0.56
export PATH=$JAVA_HOME/bin:$CATALINA_HOME/bin:$PATH

保存退出

执行命令，使文件生效

[root@localhost ~]# source /etc/profile

配置端口

Tomcat 一把默认使用端口8080，开放该端口

[root@localhost ~]# iptables -I INPUT -p tcp --dport 8080 -j ACCEPT
[root@localhost ~]# service iptables save #保存
[root@localhost ~]# service iptables restart #重启防火墙使得立即生效

启动 Tomcat，访问

[root@localhost ~]# startup.sh
[root@localhost ~]# ip addr #查看登陆 Linux IP 地址

打开浏览器，访问 http://IP:8080/

如果正常，则可访问 Tomcat

附：当前最新版本的 CentOS 7 默认不支持 iptables，需要安装该命令

可参考关于在CentOS中搭建SVN服务器
