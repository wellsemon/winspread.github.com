NetBeans 和 “Hello World”

编写 Java 程序前，先要准备好：

  • Java SE Development Kit 7
    Microsoft Windows， Solaris OS 和 Linux ： Java SE 下载列表
    Mac OS X： developer.apple.com

  • NetBeans IDE
    适合所有平台 NetBeans IDE 下载列表

附：NetBeans IDE 可以用于多种编程语言，可以选择支持一种语言或所有语言的安装包

创建应用

  • 创建 IDE 项目
    不用考虑命令行开发中的配置问题

  • 编辑源文件
    创建IDE项目时，一个基本的源文件自动创建

  • 编译源文件
    调用 javac 编译命令，由 .java 生成 .class （字节码文件）
    JVM 可以解析 .class 文件

  • 运行项目
    调用 java 启动命令，运行项目

附：
1、源文件是一个编译单元。编译该文件，可生成若干字节码文件

2、详见Java白皮书

3、NetBeans 文档

Windows 和 “Hello World”

若不适用任何 IDE，则需要准备以下工具：

  • Java SE 下载列表

  • 任一编辑工具
    如：NotePad++， JBuilder， EditPlus 和 文本编辑器

创建应用

  • 创建 .java 文件
  • 编辑 .java 文件
  • 命令行 javac 编译 .java
  • 命令行 java 运行 .class 文件

附：
1、详见Java 白皮书

Solaris OS and Linux 和 “Hello World”

类似 Windows 和 “Hello World”

不同之处：

  • J2SE 下载列表
  • 安装指导
  • cmd 替换为 terminal 终端
  • 编辑工具替换为 vi 或 emacs
